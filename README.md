# 簡易登入系統

## 介紹

簡易的登入驗證系統，輸入models/user.js裡的帳號密碼，可看到歡迎訊息，否則提示帳號或密碼錯誤

## 開始使用

1. clone 此專案到本地
2. 安裝 npm 套件
   ```bash
   npm install
   ```
3. 安裝完畢後，繼續輸入：
   ```bash
   npm run start
   ```
5. 若看見此行訊息則代表順利運行，打開瀏覽器進入到以下網址
   ```bash
   Listening on http://localhost:3000
   ```
6. 若欲暫停使用
   ```bash
   ctrl + c
   ```

## 開發工具

- node.js 14.17.6
- express 4.17.1
- express-handlebars 6.0.2
- cookie-parser 1.4.6
- uuid 8.3.2
